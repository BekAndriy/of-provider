## Installations

Install all dependencies required for the project

```
npm install
```

## Development

Launch dev server with hot-reloader

```
npm run start:dev
```

Validate and format code

```
npm run prettier
```

## Build

For production

```
npm run build

# OR

npm run build:prod
```

For development including Source-map.

```
npm run build:dev
```

## Deployment

Deploy build to the Azure Account Storage

```
npm run deploy
```
